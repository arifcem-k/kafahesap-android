# KafaHesap (Android)

KafaHesap is a simple Android application designed for quick and practical calculations.

## About
This project was developed as a standalone Android application to practice mobile development and basic application logic.

## Features
- Simple and fast calculations
- Clean and minimal UI
- Offline usage
